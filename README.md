# The AUV Project

The AUV Project is an attempt at simulating an autonomous underwater vehicle simulation. The pipeline is a combination of a signal processing, an imaging-pipeline and a perception & control pipeline. The project was inspired by Northrop Grumman's [Manta Ray Project](https://www.northropgrumman.com/what-we-do/mission-solutions/sensors/manta-ray). 

The aim of this project is to create a surveillance AUV designed to surveil sea-floor despite presence of obstructive terrain-features, sans human-intervention. 

**Notes**:
- The project is lowkey low priority, so its gonna be a while. 
- The primary implementation will be in C++, with a custom templated-library ([svr-library](https://github.com/vrsreeganesh/svr)) LibTorch.
